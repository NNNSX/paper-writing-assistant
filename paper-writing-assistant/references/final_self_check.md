# Final Self Check

Use this reference before finalizing substantial paper edits, response packages, or data-bearing revisions.

## Logic

- Main chain is complete: problem, gap, challenge, method, evidence, mechanism, boundary.
- Each contribution has supporting method, formula, figure, table, or experiment evidence.
- New material reads as part of the paper, not as a revision patch.
- Conclusion scope matches the experiments.

## Data And Claims

- Abstract numbers appear in the main-paper tables, figures, or verified experiment material.
- Contribution and introduction performance claims match the experiment section.
- Inline numeric claims have clear sources.
- `improve`, `reduce`, `outperform`, `best`, `significant`, `consistent`, `all`, and `robust` are supported by the data.
- Averages, ranks, gains, reductions, speedups, and highlights are recalculated after table changes.
- Metric units, decimal places, and direction arrows are consistent.
- Baseline protocols are marked as `reported`, `reproduced`, `official`, `estimated`, or `contextual` when needed.
- Negative results, local reversals, and protocol mismatches are not hidden.

## Writing And Format

- Terms, symbols, abbreviations, method names, dataset names, and metrics are consistent.
- Figures, tables, formulas, sections, and references use maintainable cross-references when editing LaTeX.
- Captions explain objects, settings, metrics, and key observations.
- References are real, relevant, and necessary.
- AI-like filler, unsupported strong words, and generic sentences are removed.
- Clean and tracked versions satisfy the required marking policy.

## Response Package

- Each reviewer comment is answered.
- Each promised edit or experiment appears in the manuscript.
- Response letter locations match the final document.
- Response, supplement, main paper, and PDF do not contradict each other.

## Compact Report

- References checked:
